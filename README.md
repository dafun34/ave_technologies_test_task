# ave_technologies_test_task
Тестовое задание в компанию AVE Technologies


## Краткое описание
REST API для сохранения, получения, обновления и удаления адресов пользователей. Основной идентификатор записи — номер телефона.

## Технологии
- Python 3.13
- FastAPI
- Uvicorn
- Pydantic
- Redis
- DI
- Docker
- Docker Compose

## Быстрый старт
- Используем дефолтный файл окружения для разработки:
```bash
cp .env.dev .env
```
- Запускаем сервисы через Docker Compose:
```bash
docker compose up --build
```
- API будет доступен по адресу: `http://localhost:8000`
- Документация по API будет доступна по адресу: `http://localhost:8000/docs`

## Методы API
- `POST api/v1/address/` — Создание нового адреса пользователя.
Пример запроса:
```json
{
    "phone_number": "+79061112233",
    "address": "Лавочкина"
}
```
Пример ответа:
```json
{
  "success": true,
  "result": {
    "phone_number": "+79061112233",
    "address": "Лавочкина"
  }
}
```

- `GET api/v1/address/{phone_number}` — Получение адреса пользователя по номеру телефона.
Пример ответа:
```json
{
  "success": true,
  "result": {
    "phone_number": "+79061112233",
    "address": "Лавочкина"
  }
}
```

- `PUT api/v1/address/{phone_number}` — Обновление адреса пользователя по номеру телефона.
Пример запроса:
```json
{
    "address": "Советская"
}
```
Пример ответа:
```json
{
  "success": true,
  "result": {
    "phone_number": "+79061112233",
    "address": "Советская"
  }
}
```
- `DELETE api/v1/address/{phone_number}` — Удаление адреса пользователя по номеру телефона.  
Статус ответа при успешном удалении: `204 No Content`  
Статус ответа при отсутствии записи: `404 Not Found`


## Тестирование
Создание и активация окружения
```bash
source ./.venv/bin/activate # (linux)
uv sync
```
Запуск тестов
```bash
pytest .
```
